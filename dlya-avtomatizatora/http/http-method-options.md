# HTTP method Options

есть еще HTTP method OPTIONS, он не входит в CRUD, но он есть. Используется для определения возможностей веб-сервера или параметров соединения для конкретного ресурса. В ответ серверу следует включить заголовок Allow со списком поддерживаемых методов. Также в заголовке ответа может включаться информация о поддерживаемых расширениях.&#x20;
